C++ CALENDAR APP


A Group University Project where the group was tasked with writing a C++ calendar app that consisted of:

  • Multiple Time objects
  • Multiple Event objects
  • Multiple Date objects
  • One Calendar object

The calendar Object was my responsibility and required the following to be implemented:


  • Add an entry object by reference
  • Remove an entry object by reference
  • Update an entry object matching a provided date and
    time with the details contained in a new entry object
  • Remove all entry objects from the calendar
  • Remove all entry objects with a date string matching a provided date
  • Remove all entry objects between two provided dates
  • Return the number of entries in the calendar
  • Return all entry objects between two dates
  • Return the number of entries between dates
  • Return the entry objects with a date matching a provided date
  • Return all entry objects in the calendar
  • Return an entry at a specified position in the calendar

I also implemented operator overloads and helper functions to enable more efficient operations


The mark I received for this project was 100%

C++ Date Class

As a university assignment I was tasked with creating a Date class within C++ that could:

  • Validate input
  • Be constructed in multiple ways
    • Passing nothing and creating an object with the default values of "01/01/2000
    • Passing of an individual day, month and year
    • Passing of a string that was either of the format "d/m/yyyy", "d/mm/yyyy", "dd/m/yyyy" or "dd/mm/yyyy"
    • Passing of another Date object
  • Increment the date correctly in all circumstances
  • Protect it's own data members from modification
  • Pass tests on Travis CI

The mark I received for this project was 100%
